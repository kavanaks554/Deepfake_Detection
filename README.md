# Deepfake Detection using CNN

## Overview
This project detects whether a facial image is real or fake using a Convolutional Neural Network (CNN).

## Features
- Image upload
- Real/Fake prediction
- CNN-based classification
- Streamlit web application

## Technologies Used
- Python
- TensorFlow
- OpenCV
- Streamlit
- NumPy
- Matplotlib

## How to Run

1. Install the required packages:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
streamlit run app.py
```

## Model
The model was trained on a dataset containing real and fake face images.
